# math-statistics-labs
Этот репозиторий содержит комплекс лабораторных работ по курсу "Математическая статистика", выполненных в среде Python с использованием библиотек SciPy, Matplotlib и Statsmodels. Проект демонстрирует продвинутую работу с данными —  проверки гипотез и анализа распределений до построения регрессионных моделей.


Комплекс состоит из пяти взаимосвязанных проектов:
1.	Лабораторная работа №1: "Проверка статистических гипотез"
Одновыборочные и двухвыборочные критерии (z-test, t-test, χ²-test, F-test).
Исследование распределений статистик критериев методом Монте-Карло (1000 экспериментов).
2.	Лабораторная работа №2: "Критерии согласия и однородности выборок"
Проверка соответствия данных теоретическому распределению (критерии хи-квадрат Пирсона и Колмогорова).
Проверка однородности двух выборок (критерий хи-квадрат, Колмогорова-Смирнова, знаков, Манна-Уитни).
3.	Лабораторная работа №3: "Однофакторный дисперсионный анализ (ANOVA)"
Сравнение средних значений в трёх и более группах.
Проверка однородности дисперсий (критерий Бартлетта), метод линейных контрастов и попарные сравнения (Tukey HSD).
4.	Лабораторная работа №4: "Корреляционный анализ"
Расчёт и проверка значимости коэффициентов корреляции (Пирсона, Спирмена, Кендалла).
Анализ таблиц сопряжённости, исследование корреляционных связей при трансформации переменных.
5.	Большое домашнее задание (БДЗ): "Полный цикл статистического анализа реальных данных"
Разделы: Описательная статистика, интервальное оценивание, проверка гипотез, критерии согласия (в т.ч. χ² и Харке-Бера), дисперсионный анализ, корреляционный и регрессионный анализ (простая, множественная и квадратичная регрессия).
Данные: Работа с реальным антропометрическим датасетом (обхват бедра, колена, лодыжки, рост, возраст, процент жира и др.).

Технологический стек
•	Язык программирования: Python
•	Основные библиотеки:
SciPy.stats – вся базовая и продвинутая статистика: генерация данных, проверка гипотез, критерии согласия, дисперсионный анализ.
Matplotlib – визуализация: гистограммы, диаграммы рассеяния (scatter plots), box-plot'ы, графики эмпирических и теоретических функций распределения.
Statsmodels – регрессионный анализ (включая ANOVA), пост-хок тесты (Tukey HSD).
